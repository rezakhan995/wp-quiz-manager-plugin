== Description ==

1. Create quiz from Menu > Quiz Manager > Questions.
2. Click 'Add New'.
3. Enter question title, excerpt, right answer and multiple wrong answers.
4. Add unlimited wrong answer as you like.
5. Click 'save' to publish the answer.
6. Go to 'Page' or 'Post' > Click '+' to add new Gutenberg Block > search for 'Quiz Manager Quiz'.
7. Save page / post and view in front-end.
8. All done.
